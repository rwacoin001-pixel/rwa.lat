# FE-014 个人中心与账户关闭前端备注

状态：部分完成（2026-07-12）

本轮已完成：

- 个人中心保留退出登录入口，并清理本地 Demo 会话选择状态。
- 新增 `/profile/close-account`：余额与活跃持仓提示、关闭原因、`CLOSE` 二次确认、不可逆风险声明与 Demo 请求回执。
- 安全页加入可切换的 Passkey、生物确认、当前设备与其他会话展示，以及“退出其他会话”的可操作 Demo 状态。
- 通知中心支持 All/Unread 筛选、全部标记已读、逐条标记并跳转到相应的持仓、资产或 AI 页面。
- 关闭请求不会删除用户、撤销任何真实会话或触碰资金；页面清晰说明真实生产流程需要零余额、身份复核和数据保留审查。

仍依赖后端：

- API-003：会话撤销、设备列表、Passkey 与安全事件审计。
- API-004：重新验证、区域与 KYC 资格判定。
- API-012：工单、通知订阅和偏好设置。
- 法务/隐私：账户关闭与数据保留/删除规则。

验证：构建通过后，可从 `/profile` 打开账户关闭页，或直接访问 `/profile/close-account`。
