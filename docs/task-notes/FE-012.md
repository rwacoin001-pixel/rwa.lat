# FE-012 钱包流程前端备注

状态：部分完成（2026-07-12）

本轮已完成可运行 Demo 的钱包前端预检：

- 充值、提币、转账均可选择 TRON、Ethereum、Arbitrum 网络，并展示对应的费用/结算说明。
- 提币和转账加入收款方与金额校验；提币按网络校验地址格式、扣除预估网络费用，并阻止超过可用余额的请求。
- 提币加入地址白名单确认、示例地址簿入口、风险说明和可视化预检结果。
- 通过预检后仅进入 Demo 回执状态；不会调用链上、托管商、账本或外部资金接口。
- 钱包活动记录支持 All、Wallet、Investments、Rewards 前端筛选，并为演示数据标明状态语义。

未完成/生产前置：

- API-005：真实地址簿、白名单、余额、充值监测、提币状态机、风控筛查与回调验签。
- API-007：双式记账、可追溯余额快照、手续费结算与对账。
- PARTNER-001 / PARTNER-003：合规、托管和链路合作方确认。
