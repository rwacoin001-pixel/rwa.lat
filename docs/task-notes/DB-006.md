## 实现

- 新增迁移 `1783791000000-create-polymarket-integration-model.ts`。
- 新增 7 张表：
  - `polymarket_market_mappings`：Gamma 市场、condition、内部产品和状态映射。
  - `polymarket_token_mappings`：CLOB token/outcome、tick size 和订单簿水位。
  - `polymarket_sync_watermarks`：keyset cursor、租约、连续失败和降级状态。
  - `polymarket_order_mappings`：内部订单与外部订单的唯一映射、精确价格/数量和乐观版本。
  - `polymarket_external_events`：用户/市场频道与 REST 对账事件，原始证据和 SHA-256 不可修改、不可删除。
  - `polymarket_settlement_mappings`：市场结果、赔付值、确认/争议/反转证据。
  - `polymarket_reconciliation_cases`：缺单、状态/数量/结算/赔付差异及处理闭环。
- 所有数量和价格使用精确 `numeric`，不使用浮点数；开放对账案件带部分唯一索引，避免同一差异重复创建。
