# 客服、投资争议、官方验证与反诈骗手册

版本：1.0

## 1. App 内入口

Profile → Support Center 提供账户安全、充值提现、订单、收益结算、KYC 申诉、营销与其他问题。产品详情、订单回执、交易记录和失败页均提供“提交争议”，自动带入产品 ID、订单 ID、金额、网络和时间，用户仍可编辑说明。

## 2. 工单状态与 SLA

状态：`submitted → triaged → investigating → waiting_for_user/partner → resolved → closed/appealed`。

| 级别 | 示例 | 首次响应 | 更新频率 |
|---|---|---:|---:|
| P0 | 账户接管、未授权提现、资金安全 | 15 分钟 | 每 2 小时 |
| P1 | 提现卡住、结算差异、错误限制 | 4 小时 | 每工作日 |
| P2 | KYC 补件、订单解释、费用疑问 | 1 工作日 | 每 2 工作日 |
| P3 | 建议、一般咨询 | 2 工作日 | 状态变化时 |

用户始终看到工单号、当前状态、下一次更新时间和责任方类别。涉及合作方时平台继续承担统一进度沟通，不让用户自行追踪多个机构。

## 3. 投资争议证据包

保留订单快照、报价、费用、资格结果、发行文件版本、AI 建议与确认、钱包/账本记录、合作方回执、通知和用户沟通。任何人工调整必须双人批准，以新账本凭证冲正，不修改历史记录。

## 4. 官方账号验证页

正式上线前只展示已由品牌负责人和安全负责人共同验证的域名、邮箱、X、Telegram/Discord/微信公众账号和 App 下载链接。每项显示“最后核验日期”。未填写的渠道不展示，绝不生成看似真实的占位账号。

## 5. 诈骗举报

允许提交可疑账号、URL、钱包地址、截图和交易哈希；禁止要求助记词、私钥、密码和完整验证码。举报生成 `SCAM-YYYYMM-XXXXXX`，关联安全事件、地址风险筛查、内容下架、用户通知和必要的监管/执法报告。

统一安全提示：RWA.LAT 不会通过私信索取助记词、私钥或验证码，不会要求向个人地址转账，也不会承诺保本收益。
