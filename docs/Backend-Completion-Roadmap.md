# RWA.LAT 后端完成路线

本路线以依赖和资金安全为顺序；“代码完成”不等于“生产金融功能已批准”。

## 阶段 1：生产与资金基础

- `INFRA-001`：生产配置门禁、Secret Manager、环境 IAM、密钥轮换。
- `DB-003 / API-005 / API-007`：钱包、双式账本、提现结算/退款和对账。
- `INFRA-002 / INFRA-004`：任务队列、死信、重放、指标、报警和事故开关。

## 阶段 2：资产、订单与持仓

- `DB-004`：资产、产品版本、文件、报价、订单、执行、持仓、收益和结算模型。
- `API-006`：资产目录、披露、价格来源和数据新鲜度门禁。
- `API-008`：报价、锁款、订单、执行、结算、取消和争议状态机。
- `API-009`：Polymarket 内外订单映射、回调、结算和对账。
- `API-010`：持仓、价格快照、收益、赎回和历史表现。

## 阶段 3：用户服务与智能层

- `DB-005`：通知、工单、邀请、订阅、费用、奖励和偏好。
- `API-011`：结构化 AI 建议、来源、数据时点、安全拒绝和成本审计。
- `API-012`：通知、工单、邀请、订阅和偏好 API。
- `INFRA-003`：受控对象存储、临时访问、病毒扫描和保留策略。

## 阶段 4：管理、治理与发布

- `API-013`：后台 RBAC、最小权限、四眼审批和审计导出。
- `DB-006`：Polymarket 映射、回调事件、水位和对账模型。
- `DB-007`：保留、删除、脱敏、备份和恢复演练。
- `INFRA-005`：CI/CD、迁移门槛、回滚、依赖扫描和 SBOM。
- `QA-002` 至 `QA-006`、`RELEASE-001`：状态机、安全、账本不变量和全链路生产彩排。

## 外部输入原则

供应商名称、正式 API 文档、地区/限额/审批规则可以在任务中确认；密钥只能进入 Secret Manager。缺少正式协议、法务批准或生产凭证时，对应适配器保持关闭，不使用桩数据冒充生产结果。
