## 使用规则

1. 产品、收益率、费率和可用地区必须来自后台配置，不得只写在页面文案中。
2. 对外发布前，由产品负责人、合规负责人和业务负责人共同批准对应版本。
3. 所有对外材料保留版本号、审批人、发布时间、适用地区和撤回记录。
4. 合作方名称、牌照编号、托管地址、发行主体和争议管辖地在正式上线前必须替换 Demo 占位信息。
